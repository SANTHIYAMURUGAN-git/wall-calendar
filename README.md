# 🗓️ Interactive Wall Calendar

A polished, interactive wall calendar component built with React.

## Features
- 📸 Hero image that changes each month
- 🗓️ Date range selector with visual start/end/in-between states
- 📝 Notes panel — attach notes to any date
- 🌙 Dark / Light mode toggle
- 🟡 Indian holiday markers
- 📱 Fully responsive (mobile + desktop)
- ✨ Smooth flip animation when changing months

## How to Run Locally

### Prerequisites
- Node.js v18+

### Steps
```bash
git clone https://github.com/YOURNAME/wall-calendar.git
cd wall-calendar
npm install
npm start
```
Open http://localhost:3000

## Tech Stack
- React (Create React App)
- lucide-react (icons)
- Inline CSS / CSS-in-JS styling
